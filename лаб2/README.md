## Описание

**Простейший транслятор подмножества языка Go в язык Python**, реализованный на Python.

### Поддерживаемые конструкции Go

- `package main` — обязательная конструкция Go, игнорируется при трансляции;
- объявление функций `func`;
- цикл `for` в Go-стиле (`for i := ...; i < ...; i++`);
- объявление переменных (`var` и оператор `:=`);
- вывод данных в консоль через `fmt.Println`.

### Результат работы

В результате работы транслятора формируется корректный Python-код с сохранением логики исходной программы и автоматическим управлением отступами.
